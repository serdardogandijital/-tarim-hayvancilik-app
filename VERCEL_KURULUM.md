# Vercel'e Deploy Talimatları

## Hızlı Kurulum

1. **Vercel'e gidin:** https://vercel.com
2. **GitHub ile giriş yapın**
3. **"Add New Project"** tıklayın
4. **Repository seçin:** `serdardogandijital/-tarim-hayvancilik-app`
5. **Framework Preset:** "Other" seçin
6. **Build Command:** `./build.sh` (otomatik doldurulacak)
7. **Output Directory:** `build/web` (otomatik doldurulacak)
8. **Deploy** butonuna tıklayın

## Otomatik Deploy
Her GitHub push'unda otomatik olarak deploy edilir.

## Ücretsiz Özellikler
- ✅ Sınırsız bandwidth
- ✅ SSL sertifikası (HTTPS)
- ✅ Özel domain desteği
- ✅ Otomatik CI/CD
- ✅ Global CDN

## Site URL
Deploy sonrası Vercel size otomatik bir URL verecek:
`https://-tarim-hayvancilik-app.vercel.app`

Özel domain de ekleyebilirsiniz.
