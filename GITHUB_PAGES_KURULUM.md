# GitHub Pages Kurulum Talimatları

## Otomatik Kurulum
Workflow otomatik olarak çalışacak. İlk kez için:

1. GitHub'da repository'ye gidin: https://github.com/serdardogandijital/-tarim-hayvancilik-app
2. Settings > Pages bölümüne gidin
3. Source: "GitHub Actions" seçin
4. Workflow çalıştıktan sonra site aktif olacak

## Manuel Tetikleme
1. Actions sekmesine gidin
2. "Deploy Flutter Web to GitHub Pages" workflow'unu bulun
3. "Run workflow" butonuna tıklayın
